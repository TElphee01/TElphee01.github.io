How did you create TElphee.io?

Using skills discovered while taking a Web Development course at JMU, I've built a functional web platform during my summer break. Here's an embeded Spotify playlist for the Lulz!

The core website template is a free bootstrap theme created by Start Bootstrap. I embeded Google Forms into website, along with Paypals HTML donation tool. The HTML5 video script on the first page was tricky. Bootstrap help ensure device compatibility of which I've tried my best but still some viewing issues can occur depending on device and settings. I test with my updated IPhone 10XR with Firefox and Chrome.
